# CLAUDE.md

此文件为 Claude Code (claude.ai/code) 在处理本仓库代码时提供指导。

## 项目概述

这是一个2026年个人年度计划项目。核心内容定义在 `2026_Q2_个人计划.md` 中，包含8个维度的目标：人际支点、工作与专业能力、AI方向、市场嗅觉、个人健康、个人技能与兴趣、财务、认知提升（阅读）。

## 文件结构

```
2026年度计划/
├── CLAUDE.md              # 本文件 - 开发指导
├── 2026_Q2_个人计划.md    # 核心计划内容（Q2目标与KPI）
├── 2026计划.xlsx          # 详细计划数据的Excel表格
├── 工作OKR.jpg            # OKR框架可视化参考
└── .claude/settings.local.json # 本地配置
```

## 关键文件

### 2026_Q2_个人计划.md
Q2目标的事实来源。包含8个维度，每个维度有：愿景句、Q2杠杆成果、关键指标、每周MIT举例。

### 2026计划.xlsx
详细的电子表格，包含量化跟踪数据、时间线和分解项。

## 开发任务

### HTML计划构建器
创建 `plan.html` 文件，要求：
- 以交互式网页格式渲染 `2026_Q2_个人计划.md` 的内容
- 包含复选框，跟踪所有8个维度的任务完成情况
- 支持编辑和更新计划项
- 使用localStorage持久化状态
- 移动端响应式设计

## 构建与开发

无需构建系统，直接编辑文件即可。

### 手动验证
1. 在浏览器中打开 `plan.html`
2. 检查复选框、编辑字段等交互元素
3. 验证localStorage持久化
4. 测试移动端响应式行为

## 架构说明

### 数据流向
1. **来源**: `2026_Q2_个人计划.md`（Markdown）
2. **展示**: `plan.html`（交互式网页）
3. **跟踪**: localStorage + `2026计划.xlsx`

### 交互模型
- 复选框跟踪每周MIT完成情况
- 可编辑字段允许计划更新
- 状态持久化支持跨会话跟踪

## 本地开发

- 无Node.js/npm依赖
- 原生HTML/CSS/JavaScript
- localStorage存储
- UTF-8编码支持中文
