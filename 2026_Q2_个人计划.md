# 2026 Q2 个人计划（5月–7月）

---

## 维度1：人际支点

| 项目 | 内容 |
|------|------|
| **愿景句** | 在上海建立至少2到3个真实的线下连接，同时把现有的远程友谊维护得更稳定 |
| **Q2杠杆成果** | 参加至少3次线下活动或社群（摄影、徒步、读书会等兴趣类优先）；每月和上海以外的朋友做一次真正的深度通话，不只是日常闲聊 |
| **关键指标** | Q2结束时，有至少1个在上海认识的新联系人，见过不止一次；远程朋友的联系频率没有下降 |
| **每周MIT举例** | ① 搜索本周上海一个感兴趣的线下活动并报名 ② 给其中一个远程朋友发一条真实的、不是闲聊的消息 ③ 做一次独处的"支点活动"（拍照、走进自然等），记录感受 |

## 维度2：工作与专业能力

> ⚠️ 待补充：后续将根据 TikTok 本季度 OKR 进一步完善本维度内容。

| 项目 | 内容 |
|------|------|
| **愿景句** | 在TikTok岗位上有一个可量化的、被认可的专业成果 |
| **Q2杠杆成果** | 完成一个在工作中可以被拿出来说的项目或改进，有具体的数据或反馈支撑 |
| **关键指标** | 能用一两句话清楚描述"我Q2在工作上做了什么、带来了什么结果" |
| **每周MIT举例** | ① 识别本周工作中一个可以主动推进的改进点 ② 记录一个有意思的用户行为或数据现象 ③ 和一个同事进行一次超出日常协作的真实交流 |

## 维度3：AI方向（从需求出发）

| 项目 | 内容 |
|------|------|
| **愿景句** | 完成一个从真实需求出发、有人愿意用的AI小产品 |
| **Q2杠杆成果** | 识别一个具体的、可验证的需求，并完成最小可用版本的交付——哪怕只有三五个人在用 |
| **关键指标** | 有至少一个真实用户（不是自己）使用过这个东西，并给出反馈 |
| **每周MIT举例** | ① 本周观察或记录一个"这个需求好像没有被很好满足"的场景 ② 推进当前项目的一个具体功能模块 ③ 给一个潜在用户展示或描述这个想法，收集反应 |

## 维度4：市场嗅觉

| 项目 | 内容 |
|------|------|
| **愿景句** | 建立一套属于自己的需求发现和验证习惯，让它变成日常而不是偶发 |
| **Q2杠杆成果** | 完成至少3次完整的"发现需求→验证→记录"闭环，更新需求追踪表；在塔罗或AI工具方向完成一次有深度的竞品研究 |
| **关键指标** | 需求追踪表中有不少于**6条**经过验证的记录；能说清楚在某个关注方向上，真实用户在用什么、为什么用、缺什么 |
| **每周MIT举例** | ① 记录本周观察到的一个市场信号（小红书、身边人、工作场景均可） ② 对其中一条做简单验证（搜索、问人、看数据） ③ 更新需求追踪表，写下结论而不只是现象 |

## 维度5：个人健康

| 项目 | 内容 |
|------|------|
| **愿景句** | 体重和肌肉量有可见的提升，建立稳定的训练习惯 |
| **Q2杠杆成果** | 制定可执行的增肌饮食和训练计划并执行满6周；每周完成2次力量训练，记录体重变化 |
| **关键指标** | Q2结束时体重较现在增加2到3公斤；每周力量训练完成率≥80%；体重变化每两周记录一次，趋势持续向上 |
| **每周MIT举例** | ① 完成2次力量训练（记录动作、组数、重量） ② 记录本周饮食蛋白质摄入是否达标 ③ 每两周测量并记录一次体重 |

## 维度6：个人技能与兴趣

| 项目 | 内容 |
|------|------|
| **愿景句** | 学会自由泳基础动作；让摄影或写作成为一个真正有积累感的出口 |
| **Q2杠杆成果** | 报名游泳课并完成至少8次课，能独立完成25米自由泳；完成至少10次有意识的拍摄练习，形成小型作品集；写至少6篇反思性文字 |
| **关键指标** | Q2结束时能独立完成25米自由泳不停歇；能挑出3张自己满意的照片；写作习惯每两周至少一次 |
| **每周MIT举例** | ① 完成一次游泳课或自主练习 ② 带着一个明确主题出去拍一次 ③ 写一段这周的观察或感受，不少于200字 |

## 维度7：财务

| 项目 | 内容 |
|------|------|
| **愿景句** | 储蓄率稳步提升；副业收入开始从"以时间换钱"向"用杠杆换钱"过渡；投资知识框架初步建立 |
| **Q2杠杆成果** | 坚持记账整个Q2；梳理现有副业结构，识别至少一个可以"做一次、持续产生价值"的杠杆化方向并完成初步验证；读完一本投资入门书并输出自己的框架笔记 |
| **关键指标** | Q2储蓄率较现在提升；至少形成一个杠杆化副业方向的具体方案（不要求完成，要求想清楚）；能用自己的话解释3个投资基础概念 |
| **每周MIT举例** | ① 记录本周收支 ② 思考或推进一个"可复用、不完全依赖时间投入"的副业模块 ③ 读投资相关内容20分钟并记录一个新认知 |

## 维度8：认知提升（阅读）

| 项目 | 内容 |
|------|------|
| **愿景句** | 重新建立稳定的阅读习惯，让阅读成为撬动其他目标的杠杆 |
| **Q2杠杆成果** | Q2完成6本书（每两周1本）；阅读方向聚焦市场与商业认知、AI与技术趋势、心理与自我认知三个领域，每本书输出一段核心收获笔记 |
| **关键指标** | 6本书完成率100%；每本书有不少于200字的笔记，明确写出"这本书如何影响了我的某个具体目标或决策" |
| **每周MIT举例** | ① 每日阅读30分钟，标注关键观点 ② 周末完成当周阅读内容的简短笔记（核心观点+个人关联） ③ 每两周选定下一本书，确保书单与当前最重要的目标方向对齐 |
